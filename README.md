#GOURMET FUSION FESTIVAL

##DESCRIPTION
<P>
This project is a website about a gastronomic festival.
It includes information about schedules, guest chefs, a food gallery, and contact details.
</P>

## Technologies used
- HTML5
- CSS3
- Responsive design (adapted for mobile)

## Project Structure
- **HEADER**: Logo, title, and navigation menu
- **HERO**: Main section with featured image
- **MENUS**: Table with event schedules
- **CHEFS**: Chef information and photos
- **GALLERY**: Dish images
- ** FOOTER**: Contact information and social media links

### Features
- Hamburger menu on mobile devices
- Use of Flexbox and Grid
- Responsive design
- Hover effects and modern styles
